Welcome to the Intent API

This is the API documentation for available routes.

Frontend

If you started the basic React frontend to interact with the API : http://localhost:3000

GET Endpoints

  • GET /intent – List all intents
  • GET /intent/json-ld – List all intents (JSON-LD)
  • GET /intent/<intent_id> – Get a specific intent
  • GET /intent/<intent_id>/json-ld – Get intent (JSON-LD)
  • GET /intent/<intent_id>/intentReport – Get all intent reports for an intent
  • GET /intent/<intent_id>/intentReport/<report_id> – Get specific intent report

POST Endpoints

  • POST /intent – Create a new intent
    { "author": "Alice", "content": "Do something" }

PATCH Endpoints

  • PATCH /intent/<intent_id> – Update intent content
    { "content": "Updated value" }

DELETE Endpoints

  • DELETE /intent/<intent_id> – Delete an intent
  • DELETE /intent/<intent_id>/intentReport/<report_id> – Delete an intent report

Built with Flask | API for managing intents and intent reports
